# 📅 Daily, Weekly & Monthly Routine

> **"Consistency beats intensity. 2 hours every day beats 14 hours on a weekend."**

---

## 🎯 The Offensive Security Daily System

### ⏰ Daily Routine (2-3 Hours Minimum)

```
┌─────────────────────────────────────────────────────┐
│                   DAILY SCHEDULE                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│  🌅 Morning (30 min) — CONSUME                     │
│  ├── Read 2-3 security blog posts / advisories      │
│  ├── Check Twitter/X infosec for new techniques     │
│  └── Review 1 CVE or security advisory              │
│                                                     │
│  🔥 Core Session (60-90 min) — PRACTICE             │
│  ├── Solve 1 lab / CTF challenge                    │
│  ├── OR work on a project / tool                    │
│  ├── OR practice on bug bounty target               │
│  └── OR study current phase material                │
│                                                     │
│  🌙 Evening (30 min) — REFLECT                      │
│  ├── Document what you learned                      │
│  ├── Update notes / writeups                        │
│  └── Plan tomorrow's focus                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### What to Read Daily

| Source | Type | URL |
|---|---|---|
| **PortSwigger Research** | Web security research | portswigger.net/research |
| **Project Zero Blog** | 0-day research | googleprojectzero.blogspot.com |
| **The Hacker News** | Security news | thehackernews.com |
| **Twitter/X #infosec** | Community techniques | x.com |
| **Reddit r/netsec** | Technical discussions | reddit.com/r/netsec |
| **Bleeping Computer** | Threat intel | bleepingcomputer.com |
| **Exploit-DB** | Public exploits | exploit-db.com |

### 📆 Weekly Routine

```
┌──────────────────────────────────────────────────────┐
│                   WEEKLY SCHEDULE                     │
├──────────────────────────────────────────────────────┤
│                                                      │
│  📌 Monday: Phase Study                              │
│  Focus on current phase material (theory + labs)     │
│                                                      │
│  📌 Tuesday: Lab Practice                            │
│  Solve HTB/THM machines or PortSwigger labs          │
│                                                      │
│  📌 Wednesday: Tool Building                         │
│  Work on your current project / tool                 │
│                                                      │
│  📌 Thursday: CVE Reproduction                       │
│  Pick 1 recent CVE → understand → reproduce          │
│                                                      │
│  📌 Friday: Bug Bounty / Real Targets                │
│  Hunt on real programs (when ready)                  │
│                                                      │
│  📌 Saturday: Deep Dive                              │
│  Extended 4-6 hour session on complex topic          │
│                                                      │
│  📌 Sunday: Review & Plan                            │
│  Review week's notes, plan next week, write blog     │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### Weekly Targets

- [ ] Solve 3-5 CTF challenges or lab machines
- [ ] Reproduce 1 CVE end-to-end
- [ ] Write code (tool, script, or automation) for at least 3 hours
- [ ] Read 2 technical writeups / research papers
- [ ] Document all learnings in your notes

## 🧘 Avoiding Burnout

| Principle | Action |
|---|---|
| **Consistency > Intensity** | 2 hours daily > 14 hours on weekend |
| **Rotate topics** | Switch between web, network, RE to stay fresh |
| **Celebrate wins** | Acknowledge every solved machine, every bug found |
| **Take breaks** | 1 full rest day per week minimum |
| **Connect with community** | Isolation kills motivation — join Discord/Twitter |
| **Remember WHY** | Revisit your goals and the vision from analysis.md |
