### What Makes a Great Security Portfolio

| ✅ Include | ❌ Avoid |
|---|---|
| Working code with documentation | Half-finished projects |
| README with usage examples | No documentation |
| Actual use cases from pentesting | Generic script collections |
| Clean, well-organized code | Spaghetti code |
| Original tools solving real problems | Copies of existing tools |
| Blog posts explaining your methodology | Just code dumps |
