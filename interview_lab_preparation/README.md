## Essential Tools

- **Burp Suite** (Community/Pro) - HTTP proxy, repeater, intruder
- **OWASP ZAP** - Free alternative to Burp
- **SQLMap** - Automated SQL injection
- **Nmap** - Port/service scanning
- **Gobuster/Feroxbuster** - Directory brute forcing
- **Hashcat/John** - Password cracking
- **Wireshark** - Network traffic analysis
- **Frida/Objection** - Mobile app testing
- **jadx/apktool** - Android reverse engineering
- **dnSpy/ILSpy** - .NET reverse engineering
- **Postman/curl** - API testing
