In 2026, the transition from "Grade 2" to "Expert" involves moving beyond manual testing into **Agentic Offensive Security**. This updated roadmap integrates AI as both a target and a tool, reflecting the latest threats like "wiper" attacks and AI supply chain poisoning.

### **Revised 6-Month Pentesting Roadmap (2026 Edition)**

---

### **Month 1: Modern Web & API Foundations**

* **Focus:** Mastering the "Invisible" Web.
* **Key Skills:**
* **Advanced API Pentesting:** Move beyond REST to GraphQL and gRPC, focusing on **Broken Object Level Authorization (BOLA)**.
* **Business Logic Flaws:** Learning to chain "low-severity" misconfigurations (CORS, excessive permissions) into high-impact exploits.


* **Tooling:** Burp Suite Professional with **Burp AI** for automated payload variation.

### **Month 2: Post-Exploitation & Advanced Pivoting**

* **Focus:** Enhancing your current work with **Ligolo-ng**.
* **Key Skills:**
* **Layer-3 Transparency:** Deep dive into TUN interface manipulation to route traffic as "local" across segmented VLANs.
* **Living off the Land (LotL):** Using built-in admin tools (PowerShell, Microsoft Intune) to move laterally without dropping malware.
* **Double Pivoting:** Mastering multi-hop tunnels to reach isolated "Crown Jewel" databases.



### **Month 3: AI Attack Surfaces (The "Agentic" Shift)**

* **Focus:** Learning the **OWASP Top 10 for LLMs**.
* **Key Skills:**
* **Prompt Injection:** Practicing "Direct" and "Indirect" injection to force AI agents to execute unauthorized system commands.
* **Data Exfiltration via AI:** Tricking integrated assistants into leaking PII or system prompts from their context windows.
* **RAG Poisoning:** Learning how to manipulate the "Knowledge Base" an AI uses to return malicious or false information to users.



### **Month 4: Cloud-Native & Identity Hacking**

* **Focus:** Multi-cloud misconfigurations.
* **Key Skills:**
* **IAM Escalation:** Targeting over-permissioned service accounts and "orphaned" tokens in AWS/Azure.
* **CI/CD Pipeline Attacks:** Injecting malicious code into GitHub Actions or Jenkins to automate the "wiper" style attacks seen recently.


* **Project:** Build a lab mimicking a Hospital Management System (HMIS) on the cloud and try to escalate from a "Guest" role to "Database Admin."

### **Month 5: Agentic Pentesting & Automation**

* **Focus:** Learning to orchestrate AI Agents.
* **Key Skills:**
* **Autonomous Red Teaming:** Exploring tools like **Penligent** or **Kali MCP** to automate reconnaissance while you focus on the final exploit.
* **Custom Agent Design:** Writing your own AI-powered "glue code" to link Nmap results directly into custom exploit scripts.


* **Certification Target:** Look into the **Certified Offensive AI Security Professional (COASP)**.

### **Month 6: The "Expert" Capstone & Real-World VDP**

* **Focus:** Translation of technical impact.
* **Key Skills:**
* **Proof-of-Concept (PoC) Generation:** Creating non-destructive PoCs (e.g., triggering a safe `echo` command instead of a delete).
* **Reporting for Executives:** Learning to explain how a "minor" AI hallucination leads to a "major" financial breach.


* **Activity:** Join **OpenBugBounty** or a similar VDP to build a public portfolio of validated vulnerabilities in real-world systems.
