## 🔴 MONTH 1 — Web Application Security Deep Dive
**Theme:** "Go from knowing OWASP Top 10 to exploiting them reliably"

**Why this month matters:** Web app pentesting is your fastest path to employment AND bug bounty. Your dev background makes this your strongest area. Go deep, not wide.

### Weekly Breakdown

| Week | Focus | Specific Tasks | Hours/Day |
|------|-------|----------------|----------|
| Week 1 | PortSwigger SQLi + XSS | Complete ALL apprentice + practitioner SQLi labs. Complete ALL XSS labs (reflected, stored, DOM). Document bypass techniques. | 6-7h |
| Week 2 | PortSwigger Auth + Access Control | Authentication vulnerabilities labs (all). Access control labs (all). IDOR exploitation patterns. | 6-7h |
| Week 3 | PortSwigger SSRF + File Upload + SSTI | Server-side attacks cluster. File upload bypass techniques. Template injection basics. | 6-7h |
| Week 4 | Integration Week | Test DVWA at all levels with new knowledge. Write 2 professional pentest reports. Set up bug bounty recon toolchain. Start 1st blog post. | 6-7h |

### Month 1 Deliverables
- [ ] 40+ PortSwigger labs completed
- [ ] 2 web app pentest reports (DVWA or Juice Shop)
- [ ] Bug bounty recon toolchain set up (subfinder, httpx, nuclei, gau)
- [ ] 1 blog post published (Medium or GitHub Pages)
- [ ] Daily routine followed for 25+ days
- [ ] LinkedIn profile updated, 20+ security professionals connected

### Financial Action
- Calculate exact monthly expenses → write it down
- List potential freelance dev clients on Upwork
- Apply for 2-3 small freelance dev gigs as safety net

## 🟢 MONTH 5 — Application Security: Your Developer Superpower
**Theme:** "This is where your dev background becomes your #1 selling point."

**Why AppSec now:** By Month 5, your offensive skills are solid. Now add the skill that NO OTHER pentesting candidate from a non-dev background can offer: secure code review.

### Weekly Breakdown

| Week | Focus | Specific Tasks | Hours/Day |
|------|-------|----------------|----------|
| Week 17 | Source Code Review Fundamentals | OWASP Code Review Guide (read Chapters 1-8). Practice finding SQLi, XSS, IDOR in code. Use Semgrep for automated scanning. | 6h |
| Week 18 | Real-World Code Review | Pick 2 open-source web apps (Django/Flask/Express). Manual code review: find authentication flaws, injection points, logic bugs. Document findings as formal reports. | 6-7h |
| Week 19 | Build Your Own Vulnerable App | Use Spring Boot (your strength!) to build a deliberately vulnerable API. Include: SQLi, IDOR, broken auth, mass assignment. Write pentest report for it. | 6-7h |
| Week 20 | Burp Extension + Automation | Build a custom Burp extension (Java or Python). OR build a security scanning tool in Python. Push to GitHub with documentation. | 5-6h |

### Month 5 Deliverables
- [ ] 2 code review reports on open-source projects
- [ ] 1 vulnerable app built (Spring Boot) + pentest report
- [ ] 1 Burp extension OR security tool on GitHub
- [ ] Blog post: "Code Review for Security: A Developer's Perspective"
- [ ] Start job applications — 5 per week
- [ ] Resume v2 finalized with all project experience
- [ ] eJPT exam scheduled (if budget allows)

## ⚫ MONTH 12 — Endgame
**Theme:** "Evaluate, decide, plan Year 2."

### Monthly Focus

| Outcome | Action |
|---------|--------|
| If you HAVE a job offer | Negotiate salary (never accept first offer). Plan 90-day learning plan for the new role. |
| If you have INTERVIEWS but no offer | Analyze rejection reasons. Get feedback. Double down on weak areas. Continue applying. |
| If you have FEW interviews | Resume needs work. Network harder. Consider SOC analyst roles as stepping stone. Lower salary expectations temporarily. |
| If NOTHING is working | Consider: Is your resume reaching humans? Are you applying to the right roles? Is GPSC exam a better short-term path? |

### Month 12 Deliverables
- [ ] Year 1 retrospective written
- [ ] Year 2 plan drafted
- [ ] Portfolio fully updated
- [ ] Financial assessment: Can you sustain another 6 months?
- [ ] Certification plan for Year 2 (OSCP, CRTP, or cloud security cert)
- [ ] GPSC: If exam is in Year 2, create dedicated 3-month prep plan
