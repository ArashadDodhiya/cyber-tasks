# ⏰ Daily Timetables — 4 Schedule Variants

> Not every day is the same. Pick the schedule that matches your energy. The key is NEVER having a zero day.

---

## Schedule 1: 🔥 Ideal Day (High Productivity — 9-10 hours)
**Use when:** You're well-rested, motivated, and have no distractions.

| Time | Activity | Details |
|------|----------|---------|
| 06:30 - 07:00 | Wake up, freshen up | No phone for first 30 min |
| 07:00 - 07:30 | Exercise | Walk, run, or bodyweight workout |
| 07:30 - 08:00 | Breakfast + Security News | Read 2-3 articles (The Hacker News, Bleeping Computer) |
| 08:00 - 10:30 | 🔴 **DEEP WORK 1: Core Study** | Current phase topic (labs, theory, hands-on). NO distractions — phone in another room. |
| 10:30 - 10:45 | Break | Walk, stretch, hydrate |
| 10:45 - 13:00 | 🔴 **DEEP WORK 2: Hands-On Practice** | HTB machines / PortSwigger labs / Bug bounty hunting. Pure practical work. |
| 13:00 - 14:00 | Lunch + Rest | No screens. Eat properly. |
| 14:00 - 16:00 | 🟡 **BUILD SESSION** | Tool building / code review / projects / freelance dev (income). Pick ONE. |
| 16:00 - 16:15 | Break + Snack | |
| 16:15 - 17:30 | 🟢 **DOCUMENTATION** | Blog writing, writeups, update portfolio, document what you learned today. |
| 17:30 - 18:30 | 🔵 **NETWORKING + JOB HUNT** | LinkedIn activity, Discord engagement, job applications (Month 5+). |
| 18:30 - 19:30 | Dinner + Family time | Completely offline |
| 19:30 - 21:00 | 📚 **LIGHT STUDY** | Watch conference talks, read writeups, CTF practice, GPSC prep (1h). |
| 21:00 - 22:00 | Wind down | Journal 3 things learned, plan tomorrow, personal time |
| 22:00 - 06:30 | Sleep | 8.5 hours. Non-negotiable. |

**Total productive hours: ~9-10h** | **Deep focused work: ~5h** | **GPSC: ~1h**

## Schedule 2: 🛡️ Burnout-Safe Day (Sustainable — 6-7 hours)
**Use when:** You've been grinding for 5+ days straight. You feel tired but don't want a full rest day.

| Time | Activity | Details |
|------|----------|---------|
| 07:00 - 07:30 | Wake up, freshen up | Sleep an extra 30 min |
| 07:30 - 08:00 | Light exercise | Gentle walk only, no intense workout |
| 08:00 - 08:30 | Breakfast + Casual reading | Interesting security articles, NOT study material |
| 08:30 - 10:30 | 🟡 **MODERATE STUDY** | Review what you learned this week. Revise notes. Redo a lab you struggled with. |
| 10:30 - 11:00 | Long break | Tea/coffee, stretch, go outside |
| 11:00 - 12:30 | 🟡 **LIGHT PRACTICE** | Easy HTB machines / reviewed PortSwigger labs. Things you already know — build confidence. |
| 12:30 - 14:00 | Lunch + Extended rest | Nap if needed. No guilt. |
| 14:00 - 15:30 | 🟢 **CREATIVE WORK** | Blog writing (it's less intense than hacking). Portfolio updates. LinkedIn post. |
| 15:30 - 16:30 | 🔵 **COMMUNITY** | Discord, Twitter, respond to comments. Light networking. |
| 16:30 - 17:30 | 📚 **GPSC Prep** | Lighter material — objective type revision, previous papers. |
| 17:30 onwards | **FREE TIME** | Exercise, family, hobby, rest. No screens after 20:00. |
| 22:00 | Sleep | |

**Total productive hours: ~6-7h** | **Deep focused work: ~2h** | **GPSC: ~1h**

> **Rule:** Use this schedule a MAXIMUM of 2 days per week. If you need it more than that, take a full rest day instead.

## Schedule 3: ⚡ High-Focus Day (Maximum Output — 11-12 hours)
**Use when:** You have a deadline (cert exam, job interview, CTF competition). Use SPARINGLY — max 2-3 days per month.

| Time | Activity | Details |
|------|----------|---------|
| 06:00 - 06:30 | Wake up, cold water, freshen up | Start strong |
| 06:30 - 07:00 | Quick exercise + breakfast | 15 min workout, protein-heavy breakfast |
| 07:00 - 09:30 | 🔴 **DEEP WORK 1** | Hardest material first. Brain is freshest. |
| 09:30 - 09:45 | Break | |
| 09:45 - 12:00 | 🔴 **DEEP WORK 2** | Continue challenging material or shift to practice. |
| 12:00 - 12:45 | Lunch | Quick but proper meal |
| 12:45 - 15:00 | 🔴 **DEEP WORK 3** | Lab machines, CTF challenges, exam practice. |
| 15:00 - 15:30 | Break | Walk outside, sunlight, hydrate |
| 15:30 - 17:30 | 🟡 **BUILD/PRACTICE** | Tool building, report writing, code review. |
| 17:30 - 18:00 | Break + snack | |
| 18:00 - 19:30 | 🟡 **MODERATE WORK** | Documentation, blog writing, portfolio. |
| 19:30 - 20:30 | Dinner + Short rest | |
| 20:30 - 22:00 | 📚 **LIGHT REVIEW** | Review the day, revise notes, plan tomorrow. GPSC if needed. |
| 22:00 | Sleep | MUST sleep — do NOT sacrifice sleep for study |

**Total productive hours: ~11-12h** | **Deep focused work: ~7-8h** | **GPSC: ~0.5h or skip**

> ⚠️ **WARNING:** Do NOT use this schedule for more than 2-3 consecutive days. Burnout is REAL and will cost you weeks of recovery.

## Schedule 4: 🌙 Low-Energy Fallback (Minimum Viable — 3-4 hours)
**Use when:** You're sick, mentally drained, dealing with personal issues, or just having a terrible day. THE GOAL IS TO NOT HAVE A ZERO DAY.

| Time | Activity | Details |
|------|----------|---------|
| Whenever you wake up | Freshen up | No pressure on wake time |
| Morning/Afternoon | 📖 **PASSIVE LEARNING (1-2h)** | Watch a conference talk (DEFCON, Black Hat). Read a writeup. Listen to Darknet Diaries podcast. NO LABS — just absorb. |
| Afternoon | 🟢 **ONE SMALL TASK (1h)** | Solve 1 easy PortSwigger lab. OR write 500 words for a blog post. OR update 1 section of your resume. OR make 5 LinkedIn connections. Pick the EASIEST one. |
| Evening | 📚 **GPSC Light (30-60 min)** | Read 1 chapter of GPSC material. Just read, no problem-solving. |
| Rest of day | **RECOVER** | Rest, exercise gently, eat well, sleep early |

**Total productive hours: ~3-4h** | **Deep focused work: ~1h** | **GPSC: ~0.5h**

> **The only rule on a low-energy day: Do SOMETHING. Even 30 minutes counts. A 3-hour day beats a 0-hour day by infinity percent.**

## 🚨 Non-Negotiable Rules (All Schedules)

1. **Sleep 7-8 hours minimum.** Your brain consolidates learning during sleep. Cutting sleep to study more is COUNTERPRODUCTIVE.
2. **Exercise 3+ days per week.** Even a 20-minute walk counts. Physical health = mental performance.
3. **No social media in first 2 hours of waking.** Instagram/YouTube reels will eat your morning focus.
4. **One FULL rest day per week.** Preferably Sunday. No screens, no study, no guilt.
5. **3 proper meals per day.** You can't hack systems if your body isn't fueled.
6. **Journal every night.** 3 things learned + 1 thing to improve. Takes 5 minutes, prevents drift.
7. **Time-box everything.** Use Pomodoro (25 min work + 5 min break) if focus is hard.
8. **Track hours honestly.** Don't count passive YouTube watching as "study time."

## 🔄 How to Pick Your Schedule Each Day

```
Wake up → Ask yourself:

"On a scale of 1-10, what's my energy?"

8-10  → ⚡ High-Focus (only if deadline exists) or 🔥 Ideal
5-7   → 🔥 Ideal
3-4   → 🛡️ Burnout-Safe
1-2   → 🌙 Low-Energy Fallback
0     → Full Rest Day (max 1 per week)
```
