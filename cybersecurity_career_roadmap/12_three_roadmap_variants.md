# 🛋️ Three Roadmap Variants

> Not everyone has the same energy, financial situation, or risk tolerance. Pick the variant that fits YOUR reality, then commit to it.

---

## Variant 1: 🚀 Minimum Viable Roadmap (MVP)
**For when:** You need income FAST. Financial pressure is high. You can't afford to spend 12 months studying.

**Timeline:** 4-6 months to first job application, 6-9 months to employment

**Philosophy:** Get employable as fast as possible with the smallest skill set that gets you through the door. Optimize for SPEED, not depth.

### What to Focus On (ONLY these):
1. Web app pentesting (PortSwigger — 60+ labs)
2. One HTB learning path (Starting Point + 10 easy machines)
3. Report writing (2 professional reports)
4. Resume + LinkedIn + GitHub (minimum viable portfolio)
5. One certification (eJPT or Security+)
6. Job applications start Month 4

### What to SKIP:
- Cloud security
- Tool building
- Advanced AD attacks
- Bug bounty (takes too long for income)
- Blog/content creation (nice but not required)
- CTFs
- GPSC prep (focus all energy on cybersecurity job)

### Daily Schedule: 8 hours
```
5h - Core study (web pentesting + labs)
1h - Report writing / documentation
1h - Job hunting / LinkedIn
1h - Certification prep (Month 3+)
```

### Monthly Targets
| Month | Goal |
|-------|------|
| 1 | 40 PortSwigger labs, Burp Suite proficiency |
| 2 | 60+ labs, 5 HTB machines, 1 pentest report |
| 3 | 10 HTB machines, 2 reports, cert prep starts |
| 4 | Cert earned/scheduled, resume ready, start applying |
| 5 | 20+ applications, interview practice |
| 6 | 40+ applications, interviews ongoing |

### Target Roles
- SOC Analyst (easiest to get)
- Junior VAPT Analyst at consulting firm
- Application Security Analyst (if dev experience is highlighted)

### Expected Salary: ₹3-5 LPA
### Probability of Job by Month 9: 50-60%
### Risk: May end up in a lower-paying role. Can upgrade later.

## Variant 2: ⚡ Aggressive Roadmap
**For when:** You have financial runway (6-12 months of savings or family support). You want maximum skill development before entering the job market.

**Timeline:** 8-10 months of intensive preparation, targeting higher-paying roles

**Philosophy:** Build an undeniable portfolio that commands better roles and higher salary. Invest more time upfront for better long-term outcome.

### What to Focus On (Everything):
1. Deep web app pentesting (150+ PortSwigger labs)
2. Network + AD pentesting (30+ HTB machines, 3 full AD chains)
3. Application security + code review (your developer edge)
4. Tool building (3+ GitHub projects)
5. Bug bounty hunting (start Month 4, hunt seriously Month 6+)
6. Cloud security basics (flaws.cloud, CloudGoat)
7. Blog + content creation (20+ posts)
8. 1-2 certifications (eJPT + CRTP or eJPT + Security+)
9. Portfolio website
10. GPSC prep (parallel, 1h/day)

### Daily Schedule: 10-12 hours
```
6h - Core technical study + labs
2h - Tool building / code review / projects
1.5h - Blog + documentation
1h - Networking + community
1h - GPSC prep
0.5h - Exercise + health
```

### Monthly Targets
| Month | Goal |
|-------|------|
| 1-2 | 80+ PortSwigger labs, blog started, recon pipeline built |
| 3-4 | 25 HTB machines, 3 pentest reports, first VDP submission, first CTF |
| 5-6 | Code review skill, 1 security tool, vulnerable app built, eJPT earned |
| 7-8 | Cloud labs done, 2 more tools, 10+ blog posts, mock interviews |
| 9-10 | Bug bounty finding, CTF results, 40+ applications, interviews |
| 11-12 | Job landed or advanced cert (CRTP), strong portfolio, GPSC ready |

### Target Roles
- Application Security Engineer (₹6-10 LPA)
- Junior Pentester at premium consulting firm (₹5-8 LPA)
- DevSecOps Engineer (₹7-12 LPA)
- Security Engineer at product company (₹8-15 LPA)

### Expected Salary: ₹5-10 LPA
### Probability of Job by Month 12: 70-80%
### Risk: High burnout potential. Must maintain health discipline.

## Variant 3: ⚖️ Safe Balanced Roadmap (RECOMMENDED)
**For when:** You want sustainable progress without burnout. You can support 8-10 months of focused preparation with some freelance income on the side.

**Timeline:** 6-8 months of preparation + 2-4 months of active job hunting = 10-14 months total

**Philosophy:** Build strong fundamentals, create meaningful proof-of-work, maintain health, and have backup plans (GPSC + freelance dev income). This is the most realistic path for most people.

### What to Focus On:
1. Web app pentesting mastery (100+ PortSwigger labs)
2. Network + AD pentesting (25 HTB machines)
3. Application security + code review (developer advantage)
4. Tool building (2 GitHub projects)
5. Bug bounty (start Month 5, not primary income focus)
6. Cloud security basics (1 month)
7. Blog (15+ posts)
8. 1 certification (eJPT)
9. GPSC prep (1h/day as backup)
10. Freelance dev income (2-3h/day, Month 2-6 for financial stability)

### Daily Schedule: 8-9 hours
```
4-5h - Core technical study + labs
2h - Freelance dev work OR tool building (alternate days)
1h - Blog + documentation + portfolio
0.5h - Networking + LinkedIn
1h - GPSC prep
```

### Monthly Targets
| Month | Goal |
|-------|------|
| 1-2 | 60 PortSwigger labs, routine established, freelance started |
| 3-4 | 20 HTB machines, 2 pentest reports, blog with 6 posts |
| 5-6 | Code review skill, 1 tool on GitHub, eJPT earned, first job apps |
| 7-8 | Cloud basics, 2nd tool, 12+ blog posts, 30+ applications |
| 9-10 | Bug bounty attempts, interview practice, GPSC mock tests |
| 11-12 | Portfolio polished, 70+ applications, interviews, GPSC backup ready |

### Target Roles (Same as Month, in order of priority)
1. Application Security Analyst (₹4-8 LPA)
2. Junior Penetration Tester (₹3-6 LPA)
3. DevSecOps Engineer (₹5-10 LPA)
4. SOC Analyst (₹2.5-4.5 LPA) — if 1-3 don't work by Month 10

### Expected Salary: ₹4-7 LPA
### Probability of Job by Month 14: 70-80%
### Risk: Slower than aggressive, but sustainable and less likely to burn out.

## 🧠 How to Choose Your Variant

```
Answer these questions:

1. Can you survive financially for 10+ months without income?
   YES → Consider Aggressive or Balanced
   NO  → MVP or Balanced (with freelance)

2. Have you experienced burnout before?
   YES → Balanced (STRONGLY recommended)
   NO  → Any variant

3. Do you want the highest possible starting salary?
   YES → Aggressive (but understand the cost)
   NO  → MVP or Balanced

4. Is GPSC important to you?
   YES → Balanced (includes GPSC prep)
   NO  → MVP or Aggressive

5. Are you okay starting as a SOC analyst?
   YES → MVP (fastest path to employment)
   NO  → Balanced or Aggressive
```

**My recommendation for your situation:** The **Safe Balanced Roadmap** (⚖️). You get the best combination of:
- Sustainable pace (no burnout)
- GPSC as backup
- Freelance income for financial stability
- Strong enough portfolio for good roles
- Not so aggressive that you crash at Month 4
